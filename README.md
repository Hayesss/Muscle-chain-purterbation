# 临床康复图谱（Clinical Rehab Atlas）

> 人体肌肉图谱与虚拟治疗决策支持系统 — 面向物理治疗师、康复医师、运动康复师的交互式临床工具

---

## 项目概述

**临床康复图谱** 是一个整合了两个核心模块的临床决策支持平台：

| 模块 | 功能 | 用途 |
|------|------|------|
| **肌肉图谱** | 交互式人体肌肉解剖视图 + 肌肉链 + 扳机点 | 查肌肉 → 看解剖/功能/扳机点/干预 |
| **治疗模式** | 症状评估 → 鉴别诊断 → 治疗方案 → 虚拟演示 | 输入症状 → 推理诊断 → 生成方案 → 演示手法 |
| **扰动推理** | 查体输入 → 连带受累 · 干预优先级 · 姿势失衡 · 动作影响 | 选择问题肌肉/疼痛位置 → 整体推理代偿网络与治疗靶点 |

---

## 扰动推理模块（Perturbation Reasoning）

由 `muscle_perturbe` 推理引擎融合而来，提供一种「**单块/多块肌肉受累后，整体如何代偿与失衡**」的推理视角，与图谱/治疗模式互补：

- **连带受累肌肉预测**：沿协同代偿、拮抗失衡、力线传导多路径累加传导权重。
- **干预优先级**：结合问题肌肉权重 + 在连带网络中的被牵连度（图中心性近似）排序治疗靶点。
- **整体姿势 / 失衡模式**：匹配上交叉、下交叉等典型综合征。
- **动作影响**：评估受累肌肉损伤后哪些功能动作会受限。

实现说明：该模块是一个零依赖的原生 ES Module 子应用，位于 `public/perturbation/`，由 Vite 原样静态托管，不参与 TS 构建；顶部导航「扰动推理」入口跳转至 `/perturbation/index.html`，其内可一键返回图谱。采用 Apple 风格设计（SF Pro / `#0071e3` / 胶囊按钮 / 大圆角 / 结果区淡入动画）。

## 技术栈

- **React 18** + **TypeScript** + **Vite**
- **Tailwind CSS** + **shadcn/ui** 组件风格
- **Zustand** + **Immer** 状态管理
- **Framer Motion** SVG 动画
- **React Router v6** 路由
- **Recharts** 疗效图表
- **Dexie.js** (IndexedDB) 本地数据持久化
- **PWA** (Workbox) 离线可用
